# 🌤️ WeatherApp  
Модерен, тъмен и динамичен Weather App, който използва **Open‑Meteo API** за прогноза, качество на въздуха, геолокация и 5‑дневен forecast.  
Поддържа **български и английски градове**, включително кирилица чрез автоматична транслитерация.

---

## ✨ Функции

- 🔍 **Търсене на градове** (BG + EN, автоматична транслитерация)  
- 📍 **Геолокация** — показва времето за текущото местоположение  
- 🌡️ **Превключване между °C / °F**  
- 🧠 **История на последните 5 търсения**  
- 🎨 **Динамични фонови градиенти** според времето  
- 🌫️ **AQI (Air Quality Index)** с цветови индикатор  
- 💬 **AQI Tooltip** — при hover показва подробна информация  
- 📅 **5‑дневна прогноза**  
- ⚡ **Кеширане на заявките** (по‑бързо зареждане)  
- 🧊 **Glassmorphism UI** + тъмен, модерен дизайн

## 🚀 Как работи?

### 1) Потребителят въвежда град  
Пример:  
- „София“ → автоматично става „sofia“  
- „Пловдив“ → „plovdiv“  
- „Москва“ → „moskva“

### 2) Извиква се Open‑Meteo Geocoding API  
Получаваме:
- latitude  
- longitude  
- име на града  
- държава  

### 3) Зареждаме времето  
- текуща температура  
- икона  
- описание  
- вятър  
- AQI  
- 5‑дневна прогноза  

### 4) UI се обновява  
- динамичен фон  
- икони  
- tooltip  
- история  

---

## 🌈 Динамични фонове

Фонът се променя автоматично според WMO weather code:

- ☀️ Ясно → спокоен тъмен син градиент  
- ⛅ Частично облачно → синьо‑лилав  
- 🌧️ Дъжд → дълбоко кобалтово синьо  
- ❄️ Сняг → ледено синьо  
- ⛈️ Буря → мистично лилаво  

---

## 🫁 AQI Tooltip

При hover върху картата „Въздух“ се показва:

- AQI стойност  
- категория (Добро / Умерено / Лошо)  
- кратко описание  
- здравни препоръки  

---

## 📍 Геолокация

С един бутон приложението взима:

- твоите координати  
- името на града (reverse geocoding)  
- прогнозата за мястото ти
